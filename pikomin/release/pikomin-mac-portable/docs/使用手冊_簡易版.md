# Pikomin 使用手冊（超簡單版）

這份手冊是給第一次用也看得懂的人。

## 兩個終端機視窗各做什麼

- 右邊視窗：跑 `tunneld`（手機連線通道）
- 左邊視窗：跑 `./start.sh`（後端 + 前端）

簡單記法：
- 右邊管連線
- 左邊管畫面

## 開始前準備

1. iPhone 與電腦在同一個 Wi-Fi
2. iPhone 解鎖、螢幕保持亮著
3. 手機已信任這台電腦（以前連過通常就可以）

## 正確啟動流程（照順序）

### Step 1：右邊視窗先開通道

```bash
sudo $HOME/.pyenv/versions/3.13.13/bin/python -m pymobiledevice3 remote tunneld --protocol tcp
```

看到持續輸出是正常，先不要關這個視窗。

### Step 2：左邊視窗開系統

```bash
cd /Users/ggt/Documents/GitHub/susi/pikomin
PYTHON_BIN="$HOME/.pyenv/versions/3.13.13/bin/python" ./start.sh
```

看到 `All services are running.` 就表示啟動成功。

## 怎麼開始種花模式

1. 開啟 `http://localhost:5678`
2. 選擇裝置
3. 點地圖加入路徑點（至少 2 個）
4. 按「開始種花」

藍點會沿著路線移動。

## 如何刪除單一路徑點

- 右側「路線資料」每一筆都有「移除」
- 點哪一筆的「移除」，就只刪那一筆

## 如果按開始但手機不動

先看三件事：

1. 右邊 `tunneld` 視窗是不是還開著
2. 手機有沒有鎖屏
3. 頁面有沒有跳錯誤訊息（toast）

快速檢查：

```bash
curl -sS http://127.0.0.1:49151
```

如果回 `{}`，代表目前沒有抓到可用的手機 tunnel。

## 要重啟哪個視窗

- 改前端畫面：重啟左邊
- 種花不動：先看右邊，通常再重啟左邊
- 右邊出現 tunnel 錯誤：重啟右邊 `tunneld`

最保險順序：

1. 關左邊（Ctrl+C）
2. 右邊保持開啟
3. 左邊重新執行 `./start.sh`

## 關閉系統

- 左邊按 `Ctrl + C`（停止後端與前端）
- 右邊按 `Ctrl + C`（停止 tunneld）

## 一句話版

先開右邊連線，再開左邊系統；右邊別關、手機別鎖，種花就會動。
