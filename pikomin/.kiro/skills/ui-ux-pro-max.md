# Design System: iOS GPS Simulator

## Style
- Minimalism + Clean
- Light mode primary
- Accent: Blue (#3B82F6), Success: Green (#10B981), Danger: Red (#EF4444)

## Color Tokens
- Background: #FFFFFF
- Surface: #F8FAFC
- Border: #E2E8F0
- Text Primary: #0F172A
- Text Secondary: #64748B
- Text Muted: #94A3B8

## Typography
- Font: System UI / -apple-system
- Body: 14px / line-height 1.5
- Label: 12px / color #64748B
- Heading: 16px bold / color #0F172A

## Spacing
- 4px base unit, multiples of 4/8/12/16/24

## Components
- Input: border #E2E8F0, bg white, focus ring blue, radius 6px
- Button Primary: bg #3B82F6, white text, radius 6px, hover #2563EB
- Button Danger: bg #EF4444, white text
- Button Ghost: bg transparent, border #E2E8F0, text #374151
- Card/Section: bg white, border #E2E8F0, radius 8px, shadow sm
