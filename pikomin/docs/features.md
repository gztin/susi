# iOS GPS 模擬器 — 功能介紹

## 這是什麼工具？

這是一個讓你透過電腦網頁，控制 iPhone GPS 位置的工具。

把 iPhone 用 USB 接上電腦，打開瀏覽器，就可以讓 iPhone 顯示任意地點的 GPS 座標。常見用途是在 Pokemon Go 等需要移動的遊戲中，讓角色移動到你想去的地方。

---

## 功能一覽

### 裝置連線
打開網頁後，左側會自動顯示目前插著的 iPhone。如果沒有偵測到，會提示你接上 USB。

### 飛行到指定位置
輸入一組座標，iPhone 的 GPS 位置會立刻跳到那個地點。適合快速移動到遠處。

### 種花模式
設定多個路徑點，讓 iPhone GPS 自動沿著路徑移動。目前預設以 20 km/h 推送座標，也可以開啟循環模式讓它一直跑。開始時會從目前最後定位座標銜接到第一個路徑點，降低突然跳回起點造成拉回的機率。

### 最佳路線
路徑模式下加入至少 3 個花點後，可以按「最佳路線」自動重新排列節點。系統會走遍所有路徑點，最後從終點回到起點，並避免同一段路線走完後又反方向走回去。產生後會重新編號節點、畫出綠色循環線，並顯示循環總距離。

### 明信片瀏覽
開啟右側工具列的明信片圖層後，系統會透過本機後端代理 Pikmin Atlas API 載入目前地圖視窗範圍內的明信片。明信片以圖片卡片顯示，右鍵可查看名稱與座標，並可複製資訊或儲存為花點地標。

上方篩選列提供全部、廟宇、變電箱、教堂與公園類型。預設顯示全部；在全部狀態下點選單一類型會切換為只顯示該類，之後可再多選其他類型。

### 重設 GPS
按一下就能停止模擬，iPhone 恢復顯示真實位置。

---

## 使用前的準備

1. iPhone 開啟「開發者模式」：設定 → 隱私權與安全性 → 開發者模式 → 開啟
2. 用 USB 連接 iPhone 到電腦，手機上點「信任此電腦」
3. 建議關閉 iPhone 自動鎖定，避免 tunnel 斷線

---

## 限制說明

- 必須用 USB 連接，不支援 Wi-Fi
- 後端程式必須跑在電腦上，無法放到 Docker 或手機裡執行
- 拔掉 USB 後 GPS 模擬自動停止
- 種花模式依賴 tunnel 與 host bridge 穩定連線；手機鎖定、斷線或 bridge 未啟動時會停止或失敗
- 明信片圖層需要網路連線讀取 Pikmin Atlas 與圖片資源；資料會快取短時間以降低重複請求
