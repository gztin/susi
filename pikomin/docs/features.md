# iOS GPS 模擬器 — 功能介紹

## 這是什麼工具？

這是一個讓你透過電腦網頁，控制 iPhone GPS 位置的工具。

把 iPhone 用 USB 接上電腦，打開瀏覽器，就可以讓 iPhone 顯示任意地點的 GPS 座標。常見用途是在 Pokemon Go 等需要移動的遊戲中，讓角色移動到你想去的地方。

---

## 功能一覽

### 裝置連線
打開網頁後，左側會自動顯示目前插著的 iPhone。如果沒有偵測到，會提示你接上 USB。

### 飛行到指定位置
輸入一組座標，iPhone 的 GPS 位置會立刻跳到那個地點。適合快速移動到遠處。

### 種花模式
設定多個路徑點，讓 iPhone GPS 自動沿著路徑移動。可以選擇速度（步行到極速），也可以開啟循環模式讓它一直跑。

### 搖桿操作
用畫面上的虛擬搖桿即時控制移動方向，像玩遊戲手把一樣。

### 重設 GPS
按一下就能停止模擬，iPhone 恢復顯示真實位置。

---

## 使用前的準備

1. iPhone 開啟「開發者模式」：設定 → 隱私權與安全性 → 開發者模式 → 開啟
2. 用 USB 連接 iPhone 到電腦，手機上點「信任此電腦」
3. 建議關閉 iPhone 自動鎖定，避免 tunnel 斷線

---

## 限制說明

- 必須用 USB 連接，不支援 Wi-Fi
- 後端程式必須跑在電腦上，無法放到 Docker 或手機裡執行
- 拔掉 USB 後 GPS 模擬自動停止
